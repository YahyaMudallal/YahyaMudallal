## Projects

- 🔬 **ImageJ plugin** for Lipid droplet characterization (Work in progress)
- 📰 **Briefly** : a Full stack news aggregator with AI powered summaries, made in React and Golang (Work in progress)
- 🤝 **[organiz-asso](https://github.com/YahyaMudallal/organiz-asso)** : a Full stack social network to organize associations, made in React and Node.js/Express
- 🧠 **[NeuralNetwork](https://github.com/YahyaMudallal/NeuralNetwork)** : a neural network for image recognition, made in Python using only NumPy
- 📦 **[projet-ALGAV](https://github.com/YahyaMudallal/AdaptiveHuffman)** : an implementation in Python of the Adaptive Huffman coding for text compression and decompression
- 🧮 **[MODEL-project](https://github.com/YahyaMudallal/MODEL-project)** : an implementation of several polynomial multiplication algorithms in C
- ⭕ **[SmallestCircle](https://github.com/YahyaMudallal/SmallestCircle)** : an implementation of several algorithms for the smallest circle problem in Java
